# Age Check (JS)
**A dirt simple way to validate users and stay compliant worldwide.**

## Installation
`npm install @lunardesign/agecheck`
or
`yarn add @lunardesign/agecheck`

## Usage
```js
import agecheck from 'agecheck'

console.log(agecheck(13, 'us'));
```
